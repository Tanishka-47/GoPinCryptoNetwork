# GO-PIN: Global Offline Payment & Identity Network

## 🚀 Overview
GO-PIN enables offline crypto payments using QR codes and self-sovereign identity.

## 🌍 Problem
Millions lack access to banking and internet connectivity.

## 💡 Solution
Offline-first blockchain payment system with privacy and identity.

## 🔥 Features
- QR-based offline payments
- Blockchain transaction sync
- AI voice assistant
- Secure identity system

## 🛠 Tech Stack
- React.js, Tailwind CSS
- Node.js, Express.js
- Solidity, Hardhat
- Ethereum (Sepolia)
- IPFS
